Projects

Shopping List

You don't need to bring your notes and a pen for shopping anymore. This app allows you to list items and store the information in your device. It shows how many items left in the list and multiple items can be selected before deletion. React hook “useRef()” was useful to retain values across rendering.

Built with: HTML, CSS, JavaScript, React

PFC Balance Calculator

I lost over 40 pounds within a half a year. Calculating PFC balance, so called macronutrients, was important and helped me become healthy again. This app calculates the balance for people whose age is between 20 and 74. The result is based on the user's physical information and activity level. Chart.js is used to add a visual effect.

Built with: HTML, CSS, JavaScript

Tenzies

I added timer function and some CSS styles to a tutorial project from Scrimba. This app is interactive and simple to play, especially for children. I learnt React basics by building this app. Separating the stopwatch function in a separate component helped me avoid unnecessary re-rendering and made the code easy to read.

Built with: HTML, CSS, JavaScript, React
